# Coffee_Sales_Dashboard

Coffee Sales Dashboard – Excel Project

Overview

This project showcases an interactive Coffee Sales Dashboard built entirely in Excel. It demonstrates strong skills in data cleaning, analysis, and visualization, making it a complete end-to-end analytical solution.

The dashboard provides actionable insights into coffee sales performance across different products, regions, and time periods, enabling informed business decisions.

Key Features
	•	Data Cleaning & Preparation:
Cleaned raw sales data to ensure accuracy and consistency. Removed duplicates, handled missing values, and standardized formats.
	•	Pivot Tables & Pivot Charts:
Created dynamic pivot tables and charts to summarize and analyze sales trends efficiently.
	•	Slicers & Interactive Filters:
Enabled easy filtering of data by product, region, and time period to explore insights interactively.
	•	Power Pivot & KPI Measures:
Implemented Power Pivot to handle large datasets and calculated Key Performance Indicators (KPIs) for sales, growth, and performance metrics.
	•	Relevant Insights:
Designed meaningful pivot tables and charts highlighting top-selling products, sales trends, and regional performance.
	•	User-Friendly Dashboard:
Developed an intuitive dashboard layout that presents all key metrics clearly for quick decision-making.

Skills Demonstrated
	•	Advanced Excel functions (VLOOKUP, XLOOKUP, SUMIFS)
	•	Pivot Tables & Pivot Charts
	•	Slicers and interactive dashboards
	•	Power Pivot and DAX measures
	•	Data cleaning and preprocessing
	•	Analytical thinking and data visualization

Use Case

	•	Sales and marketing analysis
